# SPALF: Spectrally-Preconditioned Augmented Lagrangian on Stratified Feature Manifolds

Sparse autoencoders (SAEs) for mechanistic interpretability typically minimize a weighted sum of reconstruction error and sparsity, requiring manual coefficient tuning that conflates the fidelity-sparsity trade-off with optimizer dynamics. SPALF reformulates SAE training as constrained optimization: minimize expected $\ell_0$ sparsity subject to faithfulness, lexical drift, and orthogonality constraints, solved via an augmented Lagrangian with adaptive dual dynamics.

The decoder is partitioned into a **stratified manifold** with two strata. An *anchored* block $W_A \in \mathbb{R}^{d \times V}$, initialized from the model's unembedding matrix and constrained to remain aligned with vocabulary directions, provides lexical grounding. A *free* block $W_B \in \mathbb{R}^{d \times (F-V)}$, learned from scratch under unit-norm projection, captures residual structure orthogonal to the vocabulary subspace. The encoder is initialized via matched filtering against the OAS-regularized precision matrix, $W_{\text{enc}}^{[:V]} = \Sigma^{-1} W_{\text{vocab}}$, with the free block QR-orthogonalized against the anchored block.

Gating uses **JumpReLU with learnable per-feature log-thresholds**. The non-differentiable Heaviside function is replaced during backpropagation by a Moreau envelope proximal surrogate, yielding a $C^1$-smooth straight-through estimator with per-feature bandwidth $\gamma_j = (c_\varepsilon \cdot \text{IQR}_j)^2 / 2$ calibrated from the pre-activation distribution. Both the $\ell_0$ objective and constraint violations propagate gradients through this kernel via fused Triton operations.

The augmented Lagrangian employs the **AL-CoLe smooth penalty** $\Psi(g, y) = (\max(0, 2g+y)^2 - y^2)/4$, which provides a $C^1$ transition eliminating the classical non-smooth kink. Dual variables are updated by a **nuPI controller** with proportional damping ($\kappa_p = 0.5$, derived from the sampling-to-timescale ratio), while per-constraint penalty weights follow a **monotone CAPU** rule calibrated from initial violation magnitudes. Constraint signals are filtered through a dual-rate EMA with Adam-style bias correction: a fast channel ($\beta_f = 0.9$) drives penalty adaptation, and a slow channel ($\beta_s = 0.99$) drives dual updates. A discretization correction schedule, triggered by constraint satisfaction, ramps in a KL-divergence fidelity term that transitions the objective from Mahalanobis reconstruction to downstream distributional fidelity.

SPALF eliminates manual sparsity weighting, replacing heuristic coefficient tuning with principled dual control over a lexically grounded, spectrally preconditioned sparse autoencoder.
